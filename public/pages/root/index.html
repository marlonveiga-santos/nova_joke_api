Welcome to Nova Joke API !!!

Based on original joke api

Our endpoints are:
  • /api
  • /api/random
  • /api/ten
  • /api/[category name]/random
  • /api/[category name]/ten
  • /api/joke/[joke number]
  • /status
Our current available categories are:
  • general
  • knock-knock
  • programming

You have this page as root and one JSON version at endpoint: /api (SOON).

You can programmatically check all jokes and categories at /status endpoint.

You can override to legacy routes (original Joke Api) uncommenting the route in index.js
